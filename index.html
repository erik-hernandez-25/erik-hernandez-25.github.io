Summary:

This website will show how some weather variables changes as you move through the earth latitude. Using the data retrieved from 500 randomly selected cities using the OpenWeatherMap API that you could find in my Github Repository.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather elements and the latitude, such as: temperature, cloudiness, wind speed, and humidity. In this pages we will see those analysis
